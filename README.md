# Bartr

A Tinder-like mobile app for non-monetary trades. Swipe on items you want, get matched, and barter — no money needed.

## Quick Start

```bash
# Backend
cd backend
cp .env.example .env  # Add your Supabase credentials
npm run dev

# Frontend
cd frontend
npx expo start
```
